# 📚 Study Buddy – AI-Powered Learning Assistant

An AI-powered web application that helps students understand complex topics, summarize study material, and generate quizzes instantly using Google Gemini API.

Built as a full-stack capstone project demonstrating AI integration, modern frontend development, and serverless backend architecture.

## 📌 Features

- 💡 **Explain Mode** – Break down complex academic topics in simple, structured language  
- 📝 **Summarize Mode** – Convert lengthy notes into concise key points  
- 🧠 **Quiz Mode** – Generate practice questions for active recall  
- 💬 **Chat Mode** – Interactive AI-based study assistance  
- ⚡ Real-time response streaming  
- 🎨 Clean and responsive UI

## ⚙️ Getting Started (Local Setup)

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Suvasini911/studypla-ai.git
cd studypla-ai
```

### 2️⃣ Install Dependencies
```bash
npm install
```

### 3️⃣ Create Environment Variables

Create a `.env` file in the root directory:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_anon_key
```

### 4️⃣ Start Development Server
```bash
npm run dev
```
